# SuperRate 物流小程序

一个专业的国际物流服务小程序，提供海运、空运等多种物流解决方案。

## 功能特点

### 首页
- 轮播 Banner 展示
- 快捷功能入口
- 实时运价展示
- 热门资讯列表
- 客服咨询入口

### 消息中心
- 系统通知
- 订单状态更新
- 客服消息
- 运价订阅提醒

### 企业展示
- 企业介绍
  - 公司简介
  - 荣誉资质
  - 合作伙伴
- 优势服务
  - 海运出口
  - 空运服务
- 企业风采
  - 时间轴展示
  - 图文动态

### 个人中心
- 用户信息展示
- 订单跟踪
- 运价订阅
- 企业认证
- 专属客服

## 技术栈

- 微信小程序原生开发
- Skyline 渲染引擎
- 自定义组件
- Flex 布局
- Promise 异步处理
